## About rootkit
the *first* rootkit written in golang

---

**Usage:**
```
  rootkit show/hide <path>
  rootkit winlogon <programPath>
  rootkit unwinlogon
  rootkit disable/enable cmd/taskmgr/registry
  rootkit install
  rootkit uninstall
```
