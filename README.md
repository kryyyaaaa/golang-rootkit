## About rootkit
the *first* rootkit written in golang

---

**Usage:**
```
  rootkit <path> show/hide
  rootkit <programPath> winlogon
  rootkit unwinlogon
  rootkit disable/enable cmd/taskmgr/registry
  rootkit install
  rootkit uninstall```
