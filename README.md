## About rootkit
the *first* rootkit written in golang

---

**Usage:**
```
  rootkit show|hide <path>
  rootkit winlogon <programPath>
  rootkit unwinlogon
  rootkit disabletaskmgr
  rootkit enabletaskmgr
  rootkit disablecmd
  rootkit enablecmd
  rootkit disableregistry
  rootkit enableregistry```
